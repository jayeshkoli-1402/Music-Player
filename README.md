# Flask Music Player 

A simple web-based music player built using Flask and Jinja2.  
It plays MP3 files directly from a local folder through a clean web interface.



## Features
- Play MP3 files from a folder
- Dynamic song listing using Jinja2
- Click to play songs
- Simple and lightweight Flask backend
- No database required



## Tech Stack
- Python
- Flask
- Jinja2
- HTML, CSS, JavaScript

## Installation

1. Clone the repository
git clone (https://github.com/jayeshkoli-1402/Music-Player)

2. Go the directory by Entering: 
cd MUSICPLAYER

3. Create virtual environment

4. Activate virtual environment

5. Install dependencies by Entering:
pip install -r requirements.txt

6. run the project by Entering:
python app.py

7. follow the link in Terminal
